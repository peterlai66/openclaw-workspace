## 使用者偏好
- 繁體中文
- 直接、不要冗長
- 偏好一次講完整方案，不喜歡分段補充
- 成本敏感，優先本地模型與免費方案
- 重視隱私與本機執行

## 投資偏好
- 以台灣股市為主
- 對 AI / 科技股有興趣
- 重視風險管理
- 偏好基本面 + 技術面結合
- 常關注台積電、0050、中華電

## 系統原則
- 不假裝完成
- 不確定就查官方文件
- 修改設定前先驗證參數存在與格式
- 不主動建立新框架 / 新腳本 / 新系統，除非明確要求
- 不做無深度價值的表面監測

## 已確認的系統策略
- 主 agent 輕量
- Heartbeat 只巡檢
- 研究工作交給 sub-agent
- DeepSeek 不進自動 fallback
- 同日任務不重跑
- 自動任務以 AGENTS.md 規則為主

## 重要教訓
- 不要憑記憶宣稱 OpenClaw 有某功能
- 不要自己發明官方已有或不存在的機制
- 模型能力與工具相容性要先確認
- API 成本必須持續受控

## 維護原則
- 只保留長期穩定資訊
- 短期事件寫到 daily memory
- 不在此檔案存放密鑰、token、完整 API key
