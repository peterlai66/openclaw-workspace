## 🛠️ **承諾管理流程**

### 1. 新承諾創建
```bash
# 當做出新承諾時
echo "- $(date '+%Y-%m-%d %H:%M') | 承諾內容 | 預計完成 | 進行中 | 負責人" >> current-promises.md
```

### 2. 承諾狀態更新
```bash
# 完成時更新
sed -i "s/| 進行中 |/| $(date '+%Y-%m-%d %H:%M') | ✅ 完成 |/" current-promises.md
```

### 3. 每日檢查
```bash
# 每天檢查未完成承諾
grep "進行中\|未開始" current-promises.md
```

### 4. 每週清理
```bash
# 每週清理已完成承諾
grep -v "✅ 完成" current-promises.md > current-promises-new.md
mv current-promises-new.md current-promises.md
```
